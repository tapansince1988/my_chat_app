# my_chat_app

This is a Coding Assignment to create a chat app for a demo.

### Description

- Create a simple message sharing application in Python .

- Application should be using web socket to communicate.

- Application should be able to share Video/Images .

### Nice To Have

- Automatically play the media once received

### Note To developer

- Use coding best practices.

- Any library or technology can be used for UI.

- Application can be hosted locally or in any server.

- Developer should be prepared to provide a demo on the output.

- Developer can share the components or files in GIT/Google drive once completed.
